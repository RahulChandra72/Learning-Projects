No frameworks, no build tools — just open `index.html` and it works.

## 🚀 How to Deploy

### Option 1: Netlify (Free — Recommended)
1. Go to https://netlify.com and sign up
2. Drag & drop the `Project_one` folder onto the Netlify dashboard
3. Your site goes live instantly with a free URL
4. Connect a custom domain in Site Settings → Domain Management

### Option 2: Vercel (Free)
1. Go to https://vercel.com and sign up
2. Install Vercel CLI: `npm i -g vercel`
3. Run `vercel` inside the `Project_one` folder
4. Follow prompts — your site is live

### Option 3: GitHub Pages (Free)
1. Push the `Project_one` folder contents to a GitHub repo
2. Go to Settings → Pages → Source: main branch / root
3. Your site will be at `https://yourusername.github.io/repo-name`

### Option 4: Hostinger / GoDaddy / cPanel Hosting
1. Log in to your hosting control panel
2. Open File Manager → `public_html`
3. Upload all files from `Project_one/` into `public_html/`
4. Your site is live at your domain

## ✏️ Customization

- **Update your GitHub/LinkedIn links**: Search for `href="https://linkedin.com"` and `href="https://github.com"` in `index.html` and replace with your actual profile URLs.
- **Add a resume PDF**: Place your resume as `assets/Rahul_Chandra_Resume.pdf` for the download button to work.
- **Blog posts**: Replace the placeholder blog cards in `index.html` with your real posts.
- **Project links**: Update the `href="#"` in project cards with your actual GitHub/live URLs.
